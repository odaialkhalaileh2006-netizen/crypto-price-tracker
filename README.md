# Crypto Price Tracker 💰

A web application built with Express.js, Axios, and EJS that retrieves live cryptocurrency information from the CoinGecko API.

## Features

- Search for any cryptocurrency by name
- Display the current price in USD
- Display market capitalization
- Display the 24-hour price change
- Responsive user interface built with Tailwind CSS
- Error handling for invalid cryptocurrency names

## Technologies Used

- Node.js
- Express.js
- Axios
- EJS
- Tailwind CSS
- CoinGecko API

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
npm install
```

## Running the Project

Start the server:

```bash
node index.js
```

or if using Nodemon:

```bash
nodemon index.js
```

Open your browser and visit:

```
http://localhost:3000
```

## API Used

CoinGecko API
